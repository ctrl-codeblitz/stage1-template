# Problem 11 - Second Largest Unique Value

Given an array of integers with at least two distinct values, return the
second-largest unique value.

## Input Format

- First line: an integer n.
- Second line (or subsequent whitespace-separated tokens): n integers.

## Output Format

A single integer: the second-largest distinct value.

## Example

Input:
- 5
- 4 1 5 3 2

Output:
4

## Constraints

- n >= 2.
- At least two distinct values are guaranteed.

## Expected Complexity

- Time: O(n)
- Space: O(1)
