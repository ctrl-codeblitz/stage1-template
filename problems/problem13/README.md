# Problem 13 - Character Frequency Counter

Count the frequency of each character in a string.

## Input Format

A single line string s.

## Output Format

Print one line per distinct character in first-appearance order, using:
character:count

## Example

Input:
aabbc

Output:
a:2
b:2
c:1

## Constraints

- Counting is case-sensitive.

## Expected Complexity

- Time: O(n)
- Space: O(k), where k is the number of distinct characters
