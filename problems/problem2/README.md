# Problem 2 - Factorial

Compute n! for a non-negative integer n.

## Input Format

A single integer n.

## Output Format

A single integer: n!.

## Example

Input:
5

Output:
120

## Constraints

- n >= 0.
- Assume the result fits in 64-bit signed range for this exercise.

## Expected Complexity

- Time: O(n)
- Space: O(1)
