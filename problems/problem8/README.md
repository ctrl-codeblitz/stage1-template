# Problem 8 - Sum of Digits

Given a non-negative integer n, return the sum of its decimal digits.

## Input Format

A single integer n.

## Output Format

A single integer: the sum of digits of n.

## Example

Input:
493

Output:
16

## Constraints

- n >= 0.

## Expected Complexity

- Time: O(number of digits)
- Space: O(1)
