# Problem 21 - Rotate Array by k Steps

Rotate an array to the right by k steps.

## Input Format

Expected input is:
- First line: space-separated integers for array nums.
- Second line: integer k.

## Output Format

A single line of space-separated integers: nums rotated right by k.

## Example

Input:
1 2 3 4 5
2

Output:
4 5 1 2 3

## Constraints

- k may be larger than array length.

## Expected Complexity

- Time: O(n)
- Space: O(1) preferred (O(n) acceptable for starter)
