# Problem 10 - Count Vowels

Count the number of vowels (A, E, I, O, U) in a string, case-insensitive.

## Input Format

A single line string s.

## Output Format

A single integer: the number of vowels in s.

## Example

Input:
Pillow

Output:
2

## Constraints

- Treat uppercase and lowercase vowels equally.

## Expected Complexity

- Time: O(n)
- Space: O(1)
