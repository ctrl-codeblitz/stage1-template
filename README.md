# stage1-template
Holds grading workflow, the directory structure for problems and solutions, and the README with instructions.

**Stage 1** (Problems 1–30): iterative/basic problems. Each has `starter.cpp`, `starter.py`, and `starter.java` in `problems/problemN/`, with tests in `tests/problemN/`. For Stage 2 (recursive) problems, use the stage2-template repo.
