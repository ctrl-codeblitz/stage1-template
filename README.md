# stage1-template
Holds grading workflow, the directory structure for problems and solutions, and the README with instructions.

**Stages**
- **Stage 1** (Problems 1–30): iterative/basic problems. Each has `starter.cpp`, `starter.py`, and `starter.java` in `problems/problemN/`, with tests in `tests/problemN/`.
- **Stage 2** (Problems 31–40): recursive problems (GCD, factorial, sum digits, power, Fibonacci, binary search, decimal-to-binary, max/min, subsets, palindrome). Same structure: CPP, PY, and Java starters and tests.
